Key Features

Multi-Author Collaboration

Allow multiple authors to contribute to the same lesson, ensuring diverse perspectives and content depth.

Modular Course Structure

Courses are structured into modules and lessons, allowing for flexible and focused learning experiences.

JWT-Based Secure Authentication

Robust authentication with JWT ensures secure user sessions for authors and users accessing course content.

About the Project

This project was inspired by the need for a flexible learning management platform that allows authors to contribute knowledge on various topics in a modular structure. Each course, module, and lesson can have multiple authors, ensuring content diversity and depth.

The project uses technologies like Java Spring Boot for backend functionality, JWT for secure authentication, and MariaDB for database management. Built as a part of Holberton School's Portfolio Project.
